# Prometheus Examples

This directory contains example scripts demonstrating the Prometheus quantum phase discovery framework.

## Getting Started

- `basic_usage_example.py` - Introduction to the framework
- `advanced_usage_example.py` - Advanced features and customization

- `disordered_tfim_example.py` - Disordered Transverse Field Ising Model

- `disorder_infrastructure_example.py` - Disorder averaging framework

## Running Examples

```bash
# Run a specific example
python examples/basic_usage_example.py

# Run with custom parameters
python examples/disordered_tfim_example.py --L 16 --W 0.5
```

## Prerequisites

Ensure you have installed the package:
```bash
pip install -r requirements.txt
```
